# daw-laocaonline
Primer proyecto realizado en el CFGS Desarrollo de Aplicaciones Web cuyo objetivo es desarrollar un juego de mesa en formato online utilizando solamente HTML, CSS Y JavaScript.

El diseño de la página es originales. Las imágenes y sonidos son de uso libre.

En este proyecto la parte más importante es el código JavaScript consiguiendo que el juego pueda funcionar correctamente.
